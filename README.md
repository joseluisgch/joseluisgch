# ¡Hola! Soy José Luis Galindo Chillcce

### Ingeniero Geógrafo | Especialista SIG y Ciencia de Datos Geoespaciales 🌍💻

Soy Ingeniero Geógrafo por la Universidad Nacional Mayor de San Marcos (UNMSM), especializado en planificación territorial, ciencia de datos geoespaciales y Sistemas de Información Geográfica (SIG). 

Mi trabajo se enfoca en el desarrollo de soluciones basadas en datos para optimizar la toma de decisiones territoriales, abarcando desde el análisis espacial avanzado hasta el modelamiento multiescalar de escenarios territoriales. Me apasiona la innovación tecnológica y busco aplicar constantemente herramientas emergentes como la **GeoIA**, la visualización analítica de datos y el desarrollo de geovisores y plataformas interactivas.

### 🚀 Áreas de Experiencia e Interés
*   Investigación cuantitativa y análisis de brechas de datos socioeconómicos y territoriales.
